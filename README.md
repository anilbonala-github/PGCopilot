# PGCopilot MVP

Expo React Native MVP for hostel and PG owners. The app currently uses seeded
demo data so the owner workflow can be reviewed before backend integration.

## Included flows

- Owner login demo
- Dashboard with occupancy, collections, pending rent and recent activity
- Room and bed status management
- Tenant directory and new admission form
- Monthly rent status and payment tracking
- Food and expense overview
- Occupancy, pending rent and profit-and-loss reports

## Run locally

```powershell
npm install
npm run web
```

For a phone preview, install Expo Go and run:

```powershell
npm start
```

Then scan the QR code shown by Expo.
